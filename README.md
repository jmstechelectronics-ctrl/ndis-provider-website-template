# NDIS Provider Website Template

Standalone one-page NDIS provider website template with:

- warm, participant-friendly design
- mobile-first responsive layout
- defensive config-driven content blocks
- lightweight scroll reveal animation
- local photoreal image assets

## Project Structure

- `index.html` - main template file
- `assets/ndis-template/photos/` - local image assets used by the template

## Editing

Update the JSON config block near the bottom of `index.html` to change:

- business name and contact details
- hero copy and trust points
- featured services
- about/process/FAQ content
- footer links and social placeholders

## Deployment

This project is designed to deploy directly from the repository root on static hosts like Netlify.
